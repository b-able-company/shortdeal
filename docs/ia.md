### 1. 페이지 리스트 (전체 화면 목록)

| # | 페이지명 | 설명 |
| --- | --- | --- |
| 1 | 회원가입 | 역할 선택 (제작사/바이어) + 기본 정보 입력 |
| 2 | 로그인 | 이메일/비밀번호 인증 |
| 3 | 온보딩 - 제작사 프로필 | 회사명, 로고, 국가, 장르 태그 설정 |
| 4 | 온보딩 - 바이어 프로필 | 회사명, 국가 설정 |
| 5 | 콘텐츠 브라우징 | 전체 콘텐츠 리스트 (카드형) + 검색/필터 |
| 6 | 콘텐츠 상세 | 썸네일, 시놉시스, 희망가, 오퍼 버튼 |
| 7 | 제작사 부스 (공개) | 제작사 정보 + 해당 제작사 콘텐츠 목록 |
| 8 | 오퍼 작성 폼 | 제안가, 메모, 유효기간 입력 |
| 9 | [제작사] 대시보드 - 콘텐츠 관리 | 내 콘텐츠 목록, 추가/수정/삭제 |
| 10 | [제작사] 콘텐츠 업로드/수정 | 제목, 썸네일, 시놉시스, 희망가, 링크 입력 |
| 11 | [제작사] 대시보드 - 오퍼 관리 | 받은 오퍼 목록, 상태별 필터 |
| 12 | [제작사] 오퍼 상세 | 오퍼 내용 확인 + 수락/거절 액션 |
| 13 | [바이어] 대시보드 - 내 오퍼 | 보낸 오퍼 목록 + 상태 확인 |
| 14 | [바이어] 오퍼 상세 | 오퍼 상태, 제작사 응답 확인 |
| 15 | [공통] LOI 문서함 | 확정된 LOI 목록 |
| 16 | [공통] LOI 상세 | LOI 내용 확인 + PDF 다운로드 |
| 17 | 설정 | 계정 정보, 비밀번호 변경 |
| 18 | [관리자] 대시보드 | 가입자 수, 콘텐츠 수, 오퍼/LOI 현황 모니터링 |

### 2. URL 구조

| 페이지 | URL | 비고 |
| --- | --- | --- |
| 회원가입 | `/join` | 역할 선택 포함 |
| 로그인 | `/login` |  |
| 온보딩 - 제작사 | `/onboarding/producer` |  |
| 온보딩 - 바이어 | `/onboarding/buyer` |  |
| 콘텐츠 브라우징 | `/browse` | 쿼리: `?q=`, `?genre=`, `?price_min=`, `?price_max=` |
| 콘텐츠 상세 | `/content/:contentId` |  |
| 제작사 부스 (공개) | `/booth/:slug` | SEO 친화적 slug |
| 오퍼 작성 | `/content/:contentId/offer` | 또는 모달 처리 |
| [제작사] 콘텐츠 관리 | `/studio/contents` |  |
| [제작사] 콘텐츠 업로드 | `/studio/contents/new` |  |
| [제작사] 콘텐츠 수정 | `/studio/contents/:contentId/edit` |  |
| [제작사] 오퍼 관리 | `/studio/offers` |  |
| [제작사] 오퍼 상세 | `/studio/offers/:offerId` |  |
| [바이어] 내 오퍼 | `/my/offers` |  |
| [바이어] 오퍼 상세 | `/my/offers/:offerId` |  |
| LOI 문서함 | `/loi` |  |
| LOI 상세/다운로드 | `/loi/:loiId` |  |
| 설정 | `/settings` |  |
| [관리자] 대시보드 | `/admin` | 관리자 전용 |

### 3. 역할별 접근 권한 매트릭스

| 페이지 | 비로그인 | 제작사 | 바이어 | 관리자 |
| --- | --- | --- | --- | --- |
| 회원가입 (`/join`) | ✅ | ❌ | ❌ | ❌ |
| 로그인 (`/login`) | ✅ | ❌ | ❌ | ❌ |
| 온보딩 - 제작사 | ❌ | ✅ (1회) | ❌ | ❌ |
| 온보딩 - 바이어 | ❌ | ❌ | ✅ (1회) | ❌ |
| 콘텐츠 브라우징 (`/browse`) | ✅ | ✅ | ✅ | ✅ |
| 콘텐츠 상세 (`/content/:id`) | ✅ | ✅ | ✅ | ✅ |
| 제작사 부스 (`/booth/:slug`) | ✅ | ✅ | ✅ | ✅ |
| 오퍼 작성 | ❌ | ❌ | ✅ | ❌ |
| [제작사] 콘텐츠 관리 | ❌ | ✅ (본인) | ❌ | ❌ |
| [제작사] 콘텐츠 업로드/수정 | ❌ | ✅ (본인) | ❌ | ❌ |
| [제작사] 오퍼 관리 | ❌ | ✅ (본인) | ❌ | ❌ |
| [제작사] 오퍼 상세 (수락/거절) | ❌ | ✅ (본인) | ❌ | ❌ |
| [바이어] 내 오퍼 | ❌ | ❌ | ✅ (본인) | ❌ |
| [바이어] 오퍼 상세 | ❌ | ❌ | ✅ (본인) | ❌ |
| LOI 문서함 | ❌ | ✅ (본인) | ✅ (본인) | ❌ |
| LOI 상세/다운로드 | ❌ | ✅ (관련자) | ✅ (관련자) | ❌ |
| 설정 | ❌ | ✅ | ✅ | ✅ |
| [관리자] 대시보드 (`/admin`) | ❌ | ❌ | ❌ | ✅ |

**범례**

- ✅ 접근 가능
- ❌ 접근 불가 (리다이렉트: 비로그인→로그인, 권한 없음→403 또는 홈)
- (본인): 자신의 데이터만 접근
- (관련자): 해당 거래의 당사자만 접근
